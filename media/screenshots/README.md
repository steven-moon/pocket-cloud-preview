# Screenshots

This directory will contain screenshots of PocketCloud apps and CLI output.

## Planned screenshots

- `pocketmind-chat.png` — PocketMind iOS app showing a local AI chat session
- `pocketcloudhub-macos.png` — PocketCloudHub macOS developer tools view
- `pocket-cli-verify.png` — `pocket system verify` output showing 97%+ pass rate
- `pocket-cli-local-list.png` — `pocket system local list` showing available MLX models
- `pocket-cli-schedule.png` — `pocket task list` showing scheduled and recurring tasks
- `braindeck-macos.png` — BrainDeck study notes interface

## Contributing

If you have access to a development build and want to contribute screenshots:
1. Ensure no sensitive information (API keys, file paths, personal data) is visible
2. Use device frames where possible for app screenshots
3. Use a clean terminal theme for CLI screenshots
4. Open a GitHub issue or pull request

## Format

PNG preferred. Maximum width: 1440px (2x retina = 2880px).
